# AureP

AureP 是一款面向手机端的 Emby 播放器。

它强调三件事：更顺手的移动端浏览体验、更干净的视觉设计，以及更稳定的播放链路兼容。无论你是看电影、追剧，还是播放 STRM 和 302 直链内容，AureP 都尽量把常用操作做得更直接。

## 现在可以做什么

- 保存、切换、编辑多个 Emby 服务器
- 浏览首页推荐、继续观看、资料库和搜索结果
- 查看媒体详情、演员作品、标签分类和剧照内容
- 支持电视剧分季、选集和断点续播
- 支持字幕选择、倍速切换和播放方式切换
- 支持播放手势：亮度、音量、快进和后退
- 兼容 STRM、直连、302/原链等常见播放链路
- 播放页可查看当前内核、302 状态和直链信息

## 下载与安装

当前最新版本：`0.0.7`

- 安装包：`releases/AureP-v0.0.7.apk`
- 校验文件：`releases/AureP-v0.0.7.sha256`

安装后首次打开，需要你自行添加 Emby 服务器地址、用户名和密码。本版本不内置任何默认服务器配置。

`0.0.7` 仍然是一个早期版本。后续会继续补充更多功能，也会持续完善播放、浏览和交互体验。

这次版本主要补齐播放页链路可视化信息，继续修正继续观看卡片展示，并继续优化首页、详情页和播放页布局。

## 适合谁

- 希望在手机上更舒服地浏览 Emby 媒体库的用户
- 有 STRM、网盘直链或 302 播放需求的用户
- 更在意移动端界面体验和详情页浏览体验的用户

## 当前说明

- 具体播放效果取决于服务器能力、网络环境和设备解码能力
- 不同片源、字幕和转码策略下，表现可能会有差异
- 这个仓库用于公开发版，源码与完整工程内容在私有仓库维护
