# AureP-app

`AureP-app` 是 AureP 的公开发版仓库。

当前首发版本：`0.0.1`

## 下载

- 安装包：`releases/AureP-v0.0.1.apk`
- 校验文件：`releases/AureP-v0.0.1.sha256`
- 构建元数据：`releases/output-metadata.json`

## 当前已支持

- 多服务器保存、切换、编辑和删除
- 首页推荐、继续观看、今日活动进行时
- 资料库浏览、搜索、标签分类和演员作品页
- 详情页封面、标题图、演员、剧照、标签和简介展示
- 播放页断点续播、字幕选择、倍速、播放方式切换和实时码率显示
- 播放手势控制：亮度、音量、快进和后退
- STRM、直连和 302/原链等播放链路兼容

## 说明

- 本版本不内置默认服务器配置，首次使用请自行添加 Emby 服务器。
- 具体播放效果取决于服务器能力、网络环境和设备解码能力。
