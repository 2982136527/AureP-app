# AureP

AureP 是一款面向手机端的 Emby 播放器，支持安装包下载、版本更新和持续迭代。

它主要面向想在手机上更顺手浏览和播放 Emby 内容的用户，也兼容 STRM、直链和 302 等常见播放链路。

## 现在可以做什么

- 保存、切换、编辑多个 Emby 服务器
- 浏览首页推荐、继续观看、资料库和搜索结果
- 查看媒体详情、演员作品、标签分类和剧照内容
- 支持电视剧分季、选集和断点续播
- 支持字幕选择、倍速切换和播放方式切换
- 支持播放手势：亮度、音量、快进和后退
- 兼容 STRM、直连、302/原链等常见播放链路
- 播放页可查看当前内核、302 状态和直链信息

## 下载与安装

当前最新版本：`0.0.10`

- 安装包：GitHub Release 附件 `AureP-v0.0.10.apk`
- 校验文件：GitHub Release 附件 `AureP-v0.0.10.sha256`

安装后首次打开，需要你自行添加 Emby 服务器地址、用户名和密码。本版本不内置任何默认服务器配置。

`0.0.10` 仍然是一个早期版本。后续会继续补充更多功能，也会持续完善播放、浏览和交互体验。

这次版本继续微调整体拟态 UI、播放页缓冲提示和卡片层次。如果你在使用过程中遇到问题，欢迎继续提交 Issue。

## 当前说明

- 具体播放效果取决于服务器能力、网络环境和设备解码能力
- 不同片源、字幕和转码策略下，表现可能会有差异
- 这里提供安装包下载、更新记录和问题反馈入口

## 反馈与建议

如果你在使用过程中遇到问题，或者希望 AureP 增加新的功能、继续优化某些体验，欢迎提交 Issue。

提交播放相关 Issue 时，尽量补充片源格式、视频编码、音频编码、字幕情况、播放链路和具体现象，这样更方便针对性定位和修复。
